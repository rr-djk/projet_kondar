# T-1B.2 — obstacle.py (HSV detection)

## Description
Implémenter `detect_obstacle(frame: np.ndarray) -> BoundingBox | None` qui détecte
un obstacle orange dans une frame webcam via masque HSV.

## Spécifications
- `BoundingBox` dataclass : `x: int`, `y: int`, `w: int`, `h: int` (pixels)
- Conversion BGR → HSV (OpenCV utilise BGR par défaut)
- Masque HSV avec constantes `config.py` : `HSV_ORANGE_LOW`, `HSV_ORANGE_HIGH`
- Plus grand contour → bounding box
- Aucun contour significatif → `None`
- Fonction pure : pas d'état global, pas d'effet de bord
- ArUco fallback (T-1B.2b) hors scope — déféré

## Critères d'acceptation
- Importable sans erreur
- Image avec objet orange → retourne BoundingBox
- Image sans objet orange → retourne None
- Utilise les constantes de config.py, pas de valeurs hardcodées
