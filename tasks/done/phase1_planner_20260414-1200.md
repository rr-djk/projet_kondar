# T-1B.3 — planner.py (A* pathfinding)

## Description
Implémenter `find_path(start_mm, goal_mm, obstacles) -> list[GridPoint] | None`
avec A* sur grille 50×50, timeout 150ms.

## Spécifications
- `GridPoint` dataclass : `x: int`, `y: int` (coordonnées grille)
- Grille `GRID_SIZE = (50, 50)` depuis `config.py`
- Conversion mm → grille via `WORKSPACE_MM`
- Obstacles : liste de `BoundingBox` → cellules grille bloquées
- Timeout 150ms → retourne `None` si dépassé
- Aucun chemin → `None`
- Fonction pure : pas d'état global, pas d'effet de bord
- 4-connectivité (haut, bas, gauche, droite)

## Critères d'acceptation
- Importable sans erreur
- Chemin libre → retourne liste de GridPoints
- Obstacle sur le chemin → retourne chemin contourné
- Aucun chemin possible → retourne None
- Respecte le timeout de 150ms
